#bot_mundo_conectado
Este bot do Discord é parte do meu projeto final focado em conscientização sobre mudanças climáticas. A ideia é engajar a galera que entra no servidor através do nosso site.

#Como funciona?
O usuário entra no servidor pelo link do site.

Ele pode interagir com o bot e rodar um quiz interativo sobre o clima.
Dependendo de quantas perguntas ele acertar, o bot dá um cargo, automático no servidor para recompensar o conhecimento dele.

#Tecnologias
Python
Discord.py

#Como rodar o projeto
Para testar o bot localmente, você vai precisar:

Clonar o repositório.
Instalar as dependências (ex: pip install discord.py).
Criar um arquivo para colocar o Token do seu bot.
Rodar o arquivo principal (python bot.py).
